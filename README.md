Advent of Code 2019
====================================

Just my solutions for [Advent of Code 2019](https://adventofcode.com/2019).

How to prepare environment (only once):
~~~~
> pip install --user pipx
> pipx install pipenv
> pipenv install
~~~~

How to enter virtual environment:
~~~~
> pipenv shell
~~~~

How to test:
~~~~
> pytest
~~~~

How to run:
~~~~
> python puzzles.py puzzle<DAY>_<PART>
~~~~
